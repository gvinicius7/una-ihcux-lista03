# 🧠 UNA IHC/UX - Lista 03  
## Aplicação da 1ª Heurística de Nielsen

Este projeto foi desenvolvido como atividade prática da disciplina de IHC/UX, com o objetivo de aplicar a **1ª Heurística de Jakob Nielsen: Visibilidade do Status do Sistema**.

---

## 🎯 Objetivo

Demonstrar, por meio de um programa de console em C#, como um sistema pode manter o usuário informado sobre o que está acontecendo, fornecendo feedback claro e contínuo durante o processamento.

---

## 🧩 Heurística Aplicada

### 🔹 Visibilidade do Status do Sistema
Segundo :contentReference[oaicite:0]{index=0}, o sistema deve sempre manter o usuário informado sobre o que está acontecendo, através de feedback apropriado em tempo razoável.

✔️ No projeto, isso é aplicado através de:
- Exibição de mensagens de progresso
- Atualização dinâmica no terminal
- Simulação de processamento com tempo real
- Feedback visual com cores
